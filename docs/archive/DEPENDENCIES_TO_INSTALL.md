# Dependencies to Install

Run the following command to install all required dependencies:

```bash
npm install @stripe/stripe-js @stripe/react-stripe-js qrcode jspdf @react-pdf/renderer recharts date-fns @tiptap/react @tiptap/starter-kit @upstash/ratelimit @upstash/redis
```

## Individual Packages

### Payment Processing
```bash
npm install @stripe/stripe-js @stripe/react-stripe-js
```

### QR Code & PDF Generation
```bash
npm install qrcode jspdf @react-pdf/renderer
```

### Charts & Analytics
```bash
npm install recharts date-fns
```

### Rich Text Editor
```bash
npm install @tiptap/react @tiptap/starter-kit
```

### Rate Limiting (Optional)
```bash
npm install @upstash/ratelimit @upstash/redis
```

### Email
```bash
npm install resend
```

## Dev Dependencies
```bash
npm install --save-dev @types/qrcode @playwright/test vitest @testing-library/react
```
